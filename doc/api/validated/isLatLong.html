isLatLong function

bool isLatLong (String input)

check if string input is a latlong

Implementation

bool isLatLong(String input) {
  if (!input.contains(',')) {
    return false;
  }
  var pair = input.split(',');
  return _lat.hasMatch(pair[0]) && _long.hasMatch(pair[1]);
}
